# Vue-Shop

## Project Brief

> **Project**: E-commerce Management Platform
>
> **Technology Stack**: Vue3 + TypeScript + Vue-Router + Element Plus + Axios + ECharts

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
